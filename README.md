# ChatGPT Clone

A minimal AI-powered chatbot clone built for learning and experimentation.

## Features

- AI text chat
- Modern chat UI
- Image upload support (upcoming)
- Backend API integration
- Easy to extend

## Installation

### Clone repository
```bash
git clone https://github.com/GECHCoders/chatgpt-clone.git
```

### Move into project
```bash
cd chatgpt-clone
```

### Install frontend dependencies
```bash
cd jerry-frontend
npm install
```

### Install backend dependencies
```bash
cd ../jerry-backend
npm install
```

## Running the Project

### Start backend
```bash
cd jerry-backend
npm run dev
```

### Start frontend
```bash
cd jerry-frontend
npm run dev
```
